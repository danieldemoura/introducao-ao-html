Estrutura básica do HTML

A estrutura básica da construção de uma página HTML consistem das tags: <html> <head> <body>. 

        <html> 
            <head> 
                tags de metadados
            </head> 
            <body>
                tags de conteúdo
            </body>
        </html>
  • A tag html delimita todo o conteúdo HTML, tudo que estiver dentro dessa tag vai ser um conteúdo HTML.
  • A tag head delimita a parte de informações da página, tudo que estiver dentro dessa tag não irá aparece para o usuário, as tags que estiverem dentro da tag head estaraão relacionadas a parte de metada dados e comportamento da página html.
  • A tag body é tudo relacionado ao corpo da página HTML, tudo que for colocado entre a abertura e fechamento da tag body é o que defato vai aparecer para o usuário na página.

Tags básica do HTML

Tag de cabeçalho

A tag h1 até h6 são tags de cabeçalho, onde elas possuí uma semântica, a tag h1 é a tag de primeiro nível, mais importante, já a tag h2, é de segundo nível, ou seja, é um subtitulo do título principal que é o h1, e por isso ela é um nível abaixo da tag h1, e assim por diante até chgar na tag h6, que é o último nível das tags de cabeçalho.

Tag parágrafo

A tag p é uma tag de parágrafo, ela representa e delimita um conteúdo de parágrafo

Tag de marcação de um texto

A tag mark é uma tag de marcação de texto, ela serve para marcar ou deixar alguma parte de um texto em destaque, ela á parecida com uma caneta marcador de texto.

Tag para deixar um texto pequeno

A tag small além de deixar um texto pequeno ela serve para elemento de comentário, referência no final da página.

Tag para deixar um texto em itálico

A tag i serva para deixar um texto em itálico, ela também possuí um significado semântico, não só o visual de itálico.

Tag de abreviação

A tag <abbr> é uma tag de abreviação, ela serve para explicar uma sigla ou acrõnino de uma palavra, quando o usuário posicionar o mouse em cima de uma palavra que contém uma tag abbr, vai aparece um balão explicando aquele termo, por exemplo HTML.

Tag de texto exluído

A tag <del> serve para representar um texto que foi excluído, visualmente deixa o texto com um traçado no meio, para representar que aquele texto foi excluído, exemplo: Texto Riscado.

Tag sup

A tag sup representra um texto sobrescrito, elevado, com uma posição superior a outro texto exemplo: H2

Tag sub

A tag sub é igual a tag sup, só que inverso, ela deixa o texto subscrito, em uma posição inferior, exemplo: C8

Tag de citação

A tag <blockquote> representa um elemento de uma citação, sempre que tever uma frase que for de outra pessoa que você quiser cita-lá, utilize a tag blockquote, exemplo:

Cada sonho que você deixa pra trás, é um pedaço do seu futuro que deixa de existir.

Tags depreciadas do HTML5

Tag de fonte

A tag <font>, antes utilizana no HTML 4, agora na versão mais recente do HTML 5 ficou obsoleta, e não é recomenda utiliza-lá, essa tag era responsável por estilizar texto como sua cor, exemplo: Texto Vermelho. Mas agora como o HTML 5 ficou responsável exclusivamente pela estrutura e semântica da página, qualquer estilização deve ser feita com CSS.
